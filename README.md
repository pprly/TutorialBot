# Builder Bot

Мод для автоматического строительства по туториальным схемам в Minecraft.

## Описание

Builder Bot - второй компонент системы автоматического создания видео-туториалов по строительству. Мод загружает `.ltutorial` файлы, созданные в Tutorial Schematic Mod, и автоматически строит их с помощью фейкового игрока.

## Возможности

- ✅ Загрузка .ltutorial схем
- ✅ Интерактивное размещение схемы в мире (preview)
- ✅ Поворот схемы (0°, 90°, 180°, 270°)
- ✅ Фейковый игрок-строитель
- ✅ Автоматическое строительство по слоям
- ✅ Анимации ходьбы и установки блоков
- ✅ Умное ломание мешающих блоков
- ✅ Настройка скорости строительства
- ✅ HUD с информацией о прогрессе

## Установка

1. Установите [Fabric Loader](https://fabricmc.net/) для Minecraft 1.21+
2. Установите [Fabric API](https://modrinth.com/mod/fabric-api)
3. Скопируйте `builder-bot-x.x.x.jar` в папку `mods`
4. (Опционально) Установите Tutorial Schematic Mod для создания схем

## Использование

### Загрузка схемы

```
/build load <filename>   - Загрузить .ltutorial файл
/build list              - Список доступных схем
/build info              - Информация о схеме
/build unload            - Выгрузить схему
```

Схемы находятся в `.minecraft/schematics/tutorials/`

### Размещение схемы

После загрузки схема появляется как полупрозрачный "призрак":

**Клавиатура:**
- Стрелки (← → ↑ ↓) - перемещение по X/Z
- Page Up / Page Down - перемещение по Y
- R - повернуть на 90°
- Enter - подтвердить размещение
- Escape - отменить

**Команды:**
```
/build move <north|south|east|west|up|down> [amount]
/build rotate [90|180|270]
/build confirm
```

### Строительство

```
/build start             - Начать строительство
/build pause             - Пауза
/build resume            - Продолжить
/build stop              - Остановить
/build speed <1-10>      - Скорость (1=медленно, 10=быстро)
/build skip              - Пропустить слой
/build goto <layer>      - Перейти к слою
/build status            - Текущий статус
/build layers            - Список слоёв
```

### Полный рабочий процесс

1. Создайте схему в Tutorial Schematic Mod
2. Сохраните как .ltutorial
3. Загрузите: `/build load my_house`
4. Разместите стрелками и клавишей R
5. Подтвердите: Enter или `/build confirm`
6. Запустите: `/build start`
7. Наслаждайтесь автоматическим строительством!

## Фейковый игрок

Строительство выполняет фейковый игрок "BuilderBot":

- Ходит к позиции блока (плавная анимация)
- Поворачивается к блоку
- Ломает мешающие блоки (трава, цветы, листва)
- Устанавливает блоки с правильным состоянием
- Воспроизводит звуки установки

Это именно ServerPlayerEntity, а не моб - важно для интеграции с Flashback (запись от первого лица).

## Скорость строительства

| Уровень | Задержка | Описание |
|---------|----------|----------|
| 1 | 2 сек | Очень медленно |
| 5 | 0.5 сек | Средне (по умолчанию) |
| 10 | 0.1 сек | Максимально быстро |

## Автоматическое ломание блоков

Строитель автоматически ломает:
- Траву, высокую траву
- Цветы
- Листву
- Снег
- Лианы
- Воду/лаву (если нужно)

## Интеграция с системой

Этот мод - второй компонент системы автоматического создания туториалов:

1. **Tutorial Schematic** - Создание схем с слоями ✅
2. **Builder Bot** - Автоматическая постройка ✅ (этот мод)
3. **Flashback** - Запись геймплея (в разработке)
4. **Auto Editor** - Автоматический монтаж (в разработке)
5. **Flashback Export** - Рендер финального видео (в разработке)
